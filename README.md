## GTNH社区mod ：扭曲空间科技
* 源自GTNH普通玩家的游戏内容。 
* 每个人都可以在这儿集中实现好的想法。
* 平衡性太糟糕当然不行。
## 版本需求
* GTNH-2.4.1+ 版本：
1. 需要升级GTNH-Lanthanides mod至 0.11.0+版本
* GTNH-2.4.0 版本：
1. 需要升级GTNH-Lanthanides mod至 0.11.0+版本或专门适配版本；
2. 需要升级GregTech mod至 5.09.44.03+版本
* GTNH-2.4.2 测试版本：
1. 暂时不兼容测试版
2. 暂时未兼容新的GT配方系统
